Quick Tips

Welcome to Fluro! Get started with these tips.

Getting Started

Fluro doesn't need a tedious set-up workflow to get going. Just launch the app and you're ready to go!

Settings

Use Fluro's array of settings to make it yours.

General

Use the General tab to control:

  • The scrolling speed of the LED display.
  • The colour of the display's text.
  • The font of the display's text.
  • The position of each element on the display, including the ability to hide elements completely.

Weather

Use the Weather tab to:

  • Enable/disable forecasts for the day ahead.
  • Enable/disable forecasts for the following two days.
  • Set your location for weather data.

News

Use the News tab to:

  • Set your region (for use with custom categories).
  • Set a custom category.
  • Control the maximum amount of news articles you want to display.

Reminders

Use the Reminders tab to control the maximum amount of reminders you want to display.

Custom Text

Use the Custom Text tab to set your own text to appear on the display.
